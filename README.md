# 🏨 HostelBuddy - Smart Hostel Issue Tracking System

## 📖 About The Project

**HostelBuddy** is a full-stack web platform designed to streamline communication between students and hostel administration. It replaces informal complaint methods (like verbal reports or paper registers) with a transparent, accountable, and traceable digital system.

This project was developed for a hackathon by Team Velocity.

Built with a focus on **User Experience (UX)** and **Real-time Efficiency**, HostelBuddy ensures that no student complaint goes unnoticed and helps administration track facility health with data-driven insights.

## ✨ Key Features

### 🎓 For Students
* **One-Click Issue Reporting:** Easily report plumbing, electrical, or furniture issues with categories and priority levels.
* **Real-Time Status Tracking:** Watch your complaint move from *Reported* → *In Progress* → *Resolved*.
* **Digital Notice Board:** View important announcements (water cuts, cleaning schedules) instantly.
* **Lost & Found:** dedicated section to report lost items or claim found ones.
* **Community Interaction:** Upvote issues that affect multiple people to highlight urgency.

### 🛡️ For Administration
* **Centralized Dashboard:** A bird's-eye view of all hostel operations and pending issues.
* **Role-Based Access Control:** Secure login for Students vs. Admins.
* **Smart Analytics:** Visualize average resolution time and most frequent complaints.
* **Broadcast System:** Send targeted announcements to specific hostel blocks.

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite, Tailwind CSS, Framer Motion, Lucide React
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Authentication:** Firebase Auth / JWT (Simulated for Demo)
* **Tools:** Git, GitHub, VS Code

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites
* Node.js (v16 or higher)
* MongoDB (Running locally or Atlas URL)

### Installation

**1. Clone the Repository**
```bash
git clone [https://github.com/your-username/HostelBuddy.git](https://github.com/your-username/HostelBuddy.git)
cd HostelBuddy

**2. Install Dependencies**
```bash
npm install # or pnpm install or yarn install
```

**3. Run the Development Server**
```bash
npm run dev # or pnpm dev or yarn dev
```
