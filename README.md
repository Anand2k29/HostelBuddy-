## 📖 About The Project

**HostelBuddy** is a full-stack web platform designed to streamline communication between students and hostel administration. It replaces chaotic WhatsApp groups and paper registers with a transparent, digital ecosystem.

Built for **Team Velocity**'s Hackathon submission, HostelBuddy focuses on **User Experience (UX)**, **Safety**, and **Campus Efficiency**. It ensures no complaint goes unnoticed and provides students with modern tools for daily hostel life.

## ✨ Key Features

### 🎓 For Students (The Gen Z Experience)
* **One-Click Issue Reporting:** Report plumbing or electrical issues instantly with priority levels.
* **🔒 Anti-Ragging Cell:** A secure, **anonymous** channel to report harassment. The system hides the reporter's identity from the Admin view to ensure safety.
* **🎟️ Digital Gate Pass:** Apply for leave and get an instant **QR Code E-Outpass**. No more paper slips.
* **🧺 Smart Laundry Tracker:** IoT-simulated dashboard to check if washing machines are available or in use.
* **🔍 Lost & Found:** Upload photos of lost items or browse found items with a visual gallery.
* **📢 Digital Notice Board:** View important announcements (water cuts, events) instantly.

### 🛡️ For Administration (The Command Center)
* **Admin Console:** A "God Mode" dashboard with KPI cards (Total Issues, Pending, Resolved).
* **Role-Based Access:** Distinct secure logins for **Students** and **Wardens**.
* **Smart Analytics:** Visual insights into hostel facility health.
* **Broadcast System:** Post urgent announcements that instantly notify all students.

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite, Tailwind CSS, Framer Motion
* **Icons:** Lucide React
* **Backend:** Node.js, Express.js (Mocked for Demo)
* **Database:** MongoDB / Local Storage Simulation
* **Tools:** Git, GitHub, VS Code
